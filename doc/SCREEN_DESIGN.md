## 2. 共通仕様

### 2-1. ナビゲーション

ログイン後、全画面に共通ヘッダーを表示する。

| 要素 | 内容 |
|------|------|
| ロゴ / サービス名 | 「営業日報」クリックで日報一覧へ |
| メニュー（営業担当者） | 日報一覧 |
| メニュー（上長） | 日報一覧 |
| メニュー（管理者） | 日報一覧 / 顧客マスタ / ユーザーマスタ |
| ユーザー名表示 | ログイン中のユーザー名を右上に表示 |
| ログアウトボタン | クリックでセッション破棄 → ログイン画面へ |

### 2-2. 権限エラー

権限のない画面へアクセスした場合は `403 Forbidden` ページを表示する。

### 2-3. バリデーションエラー

- フォーム送信時、入力不備があれば該当フィールドの下にエラーメッセージを赤字で表示する。
- ページ遷移はせず、入力内容を保持したままエラーを表示する。

## 3. ログイン画面 (LOGIN)

### 概要

システムへのログイン画面。未ログインユーザーが保護されたURLへアクセスした場合もこの画面にリダイレクトする。

### レイアウト

```
┌─────────────────────────────┐
│                             │
│        営業日報             │
│                             │
│  メールアドレス             │
│  [                        ] │
│                             │
│  パスワード                 │
│  [                        ] │
│                             │
│  [  ログイン  ]             │
│                             │
└─────────────────────────────┘
```

### 入力項目

| 項目名 | 種別 | 必須 | バリデーション |
|--------|------|------|----------------|
| メールアドレス | text (email) | ○ | メール形式 |
| パスワード | password | ○ | 1文字以上 |

### アクション

| アクション | 処理 |
|------------|------|
| ログインボタン押下 | 認証成功 → 日報一覧へ遷移。失敗 → 「メールアドレスまたはパスワードが正しくありません」を表示 |

## 4. 日報一覧画面 (REPORT-LIST)

### 概要

日報の一覧を表示する。ロールによって表示範囲が異なる。

- **営業担当者**：自分の日報のみ表示
- **上長**：自分と部下全員の日報を表示
- **管理者**：全ユーザーの日報を表示

### レイアウト

```
┌──────────────────────────────────────────────────────┐
│ 営業日報   日報一覧  (管理者のみ: 顧客/ユーザー) [山田太郎 ▼] │
├──────────────────────────────────────────────────────┤
│                                    [+ 今日の日報を作成] │
│                                                      │
│  絞り込み:  担当者 [全員 ▼]  期間 [____] 〜 [____]  [検索] │
│                                                      │
│  日付         担当者       訪問件数  コメント  操作   │
│  ─────────────────────────────────────────────────   │
│  2024/01/15   山田 太郎    3件       あり      [詳細] │
│  2024/01/15   鈴木 花子    2件       なし      [詳細] │
│  2024/01/14   山田 太郎    1件       あり      [詳細] │
│  ...                                                 │
│                                                      │
│  < 1 2 3 ... >                                       │
└──────────────────────────────────────────────────────┘
```

### 表示項目

| 項目名 | 内容 |
|--------|------|
| 日付 | 日報の報告日 |
| 担当者 | 日報作成者の氏名（上長・管理者のみ表示） |
| 訪問件数 | 当日の訪問記録の件数 |
| コメント | 上長コメントの有無（あり / なし） |
| 操作 | 詳細ボタン |

### 絞り込み条件

| 条件 | 対象ロール | 内容 |
|------|-----------|------|
| 担当者 | 上長・管理者 | 部下または全ユーザーから選択 |
| 期間 | 全員 | 開始日・終了日で絞り込み |

### アクション

| アクション | 処理 |
|------------|------|
| 今日の日報を作成ボタン | 日報作成画面へ遷移（営業担当者のみ表示。当日の日報が既に存在する場合は編集画面へ遷移） |
| 詳細ボタン | 日報詳細画面へ遷移 |

## 5. 日報作成画面 (REPORT-NEW)

### 概要

当日の日報を新規作成する。当日分の日報が既に存在する場合は日報編集画面へリダイレクトする。

### レイアウト

```
┌──────────────────────────────────────────────────────┐
│ 営業日報                              [山田太郎 ▼]   │
├──────────────────────────────────────────────────────┤
│ 日報作成  2024年1月15日（月）                        │
│                                                      │
│ ▼ 訪問記録                          [+ 行を追加する] │
│ ┌────────────────────────────────────────────────┐   │
│ │ No. │ 顧客名           │ 訪問内容       │ 削除 │   │
│ │  1  │ [株式会社○○ ▼] │ [          ]   │ [×] │   │
│ │  2  │ [        ▼]     │ [          ]   │ [×] │   │
│ └────────────────────────────────────────────────┘   │
│                                                      │
│ ▼ Problem（課題・相談）                              │
│ ┌────────────────────────────────────────────────┐   │
│ │                                                │   │
│ │                                                │   │
│ └────────────────────────────────────────────────┘   │
│                                                      │
│ ▼ Plan（明日やること）                               │
│ ┌────────────────────────────────────────────────┐   │
│ │                                                │   │
│ │                                                │   │
│ └────────────────────────────────────────────────┘   │
│                                                      │
│            [キャンセル]  [提出する]                  │
└──────────────────────────────────────────────────────┘
```

### 入力項目

**訪問記録（複数行）**

| 項目名 | 種別 | 必須 | バリデーション |
|--------|------|------|----------------|
| 顧客名 | セレクトボックス（顧客マスタから選択） | ○ | 選択必須 |
| 訪問内容 | テキストエリア | ○ | 1文字以上 |

**日報本文**

| 項目名 | 種別 | 必須 | バリデーション |
|--------|------|------|----------------|
| Problem（課題・相談） | テキストエリア | ○ | 1文字以上 |
| Plan（明日やること） | テキストエリア | ○ | 1文字以上 |

### アクション

| アクション | 処理 |
|------------|------|
| 行を追加するボタン | 訪問記録の入力行を1行追加（上限なし） |
| 削除ボタン（各行） | 該当行を削除（最低1行は残す） |
| 提出するボタン | バリデーション後、日報を保存 → 日報詳細画面へ遷移 |
| キャンセルボタン | 日報一覧画面へ戻る（入力内容は破棄） |

## 6. 日報編集画面 (REPORT-EDIT)

### 概要

提出済み日報を編集する。自分が作成した日報のみ編集可能。

### レイアウト

日報作成画面と同一レイアウト。既存の入力値がフォームに反映された状態で表示される。

### 入力項目

日報作成画面と同一。

### アクション

| アクション | 処理 |
|------------|------|
| 更新するボタン | バリデーション後、日報を上書き保存 → 日報詳細画面へ遷移 |
| キャンセルボタン | 日報詳細画面へ戻る（変更内容は破棄） |

---

## 7. 日報詳細画面 (REPORT-DETAIL)

### 概要

日報の内容を閲覧する。上長はこの画面からコメントを投稿・編集できる。

### レイアウト

```
┌──────────────────────────────────────────────────────┐
│ 営業日報                              [山田太郎 ▼]   │
├──────────────────────────────────────────────────────┤
│ 山田 太郎 の日報  2024年1月15日（月）      [編集する] │
│                                                      │
│ ▼ 訪問記録                                           │
│ ┌────────────────────────────────────────────────┐   │
│ │ No. │ 顧客名       │ 訪問内容                  │   │
│ │  1  │ 株式会社ABC  │ 新製品の提案を実施。       │   │
│ │  2  │ 有限会社XYZ  │ 契約更新の確認。           │   │
│ └────────────────────────────────────────────────┘   │
│                                                      │
│ ▼ Problem（課題・相談）                              │
│  ABC社の価格交渉が難航している。上長への相談が必要。 │
│                                                      │
│ ▼ Plan（明日やること）                               │
│  ABC社向けに代替プランの資料を作成する。             │
│                                                      │
│ ▼ 上長コメント                                       │
│ ┌────────────────────────────────────────────────┐   │
│ │ ABC社については先方の予算期が3月なので、        │   │
│ │ 焦らず来週再提案する方針で進めてください。      │   │
│ │                           鈴木 一郎  01/15 18:00│   │
│ └────────────────────────────────────────────────┘   │
│ ※コメント入力欄は上長ロールのみ表示                  │
│ ┌────────────────────────────────────────────────┐   │
│ │                                                │   │
│ └────────────────────────────────────────────────┘   │
│                              [コメントを保存する]    │
└──────────────────────────────────────────────────────┘
```

### 表示項目

| エリア | 項目 | 内容 |
|--------|------|------|
| ヘッダー | 作成者氏名 | 日報作成者の氏名 |
| ヘッダー | 報告日 | 日報の対象日 |
| 訪問記録 | No. / 顧客名 / 訪問内容 | 登録された訪問記録を行ごとに表示 |
| 本文 | Problem | 課題・相談の内容 |
| 本文 | Plan | 明日やること |
| コメント | コメント本文 | 上長によるコメント（存在する場合のみ表示） |
| コメント | コメント者氏名・日時 | コメント投稿者と投稿日時 |

### コメント入力エリア

上長ロールのみ表示する。

| 項目名 | 種別 | 必須 | バリデーション |
|--------|------|------|----------------|
| コメント本文 | テキストエリア | ○ | 1文字以上 |

既存コメントがある場合は本文を初期値として表示し、上書き更新できる。

### アクション

| アクション | 対象ロール | 処理 |
|------------|-----------|------|
| 編集するボタン | 営業担当者（自分の日報のみ） | 日報編集画面へ遷移 |
| コメントを保存するボタン | 上長 | コメントを保存（新規または上書き）し、画面を再表示 |

## 8. 顧客マスタ一覧画面 (CUSTOMER-LIST)

### 概要

顧客マスタの一覧を表示・管理する。管理者のみ利用可能。

### レイアウト

```
┌──────────────────────────────────────────────────────┐
│ 営業日報  日報一覧 / 顧客マスタ / ユーザーマスタ     │
├──────────────────────────────────────────────────────┤
│ 顧客マスタ                          [+ 顧客を追加する] │
│                                                      │
│  顧客名 [        ]  担当者 [全員 ▼]  [検索]         │
│                                                      │
│  顧客名         業種       担当営業    操作          │
│  ──────────────────────────────────────────────────  │
│  株式会社ABC    製造業     山田 太郎   [編集] [削除] │
│  有限会社XYZ    IT         鈴木 花子   [編集] [削除] │
│  ...                                                 │
│                                                      │
│  < 1 2 3 ... >                                       │
└──────────────────────────────────────────────────────┘
```

### 表示項目

| 項目名 | 内容 |
|--------|------|
| 顧客名 | 顧客の正式名称 |
| 業種 | 顧客の業種 |
| 担当営業 | 担当営業担当者の氏名 |
| 操作 | 編集ボタン / 削除ボタン |

### アクション

| アクション | 処理 |
|------------|------|
| 顧客を追加するボタン | 顧客マスタ編集画面（新規）へ遷移 |
| 編集ボタン | 顧客マスタ編集画面（編集）へ遷移 |
| 削除ボタン | 確認ダイアログ表示後、削除実行。訪問記録に紐付いている顧客は削除不可（エラー表示） |

## 9. 顧客マスタ編集画面 (CUSTOMER-EDIT)

### 概要

顧客情報を新規登録または編集する。管理者のみ利用可能。

### レイアウト

```
┌──────────────────────────────────────────────────────┐
│ 営業日報                                             │
├──────────────────────────────────────────────────────┤
│ 顧客マスタ編集                                       │
│                                                      │
│  顧客名 *                                            │
│  [                                               ]   │
│                                                      │
│  業種                                                │
│  [                                               ]   │
│                                                      │
│  担当営業                                            │
│  [（未設定） ▼                                   ]   │
│                                                      │
│            [キャンセル]  [保存する]                  │
└──────────────────────────────────────────────────────┘
```

### 入力項目

| 項目名 | 種別 | 必須 | バリデーション |
|--------|------|------|----------------|
| 顧客名 | テキスト | ○ | 1文字以上、100文字以内 |
| 業種 | テキスト | - | 100文字以内 |
| 担当営業 | セレクトボックス（ユーザーマスタから選択） | - | - |

### アクション

| アクション | 処理 |
|------------|------|
| 保存するボタン | バリデーション後、保存 → 顧客マスタ一覧画面へ遷移 |
| キャンセルボタン | 顧客マスタ一覧画面へ戻る |

## 10. ユーザーマスタ一覧画面 (USER-LIST)

### 概要

ユーザー（営業担当者）マスタの一覧を表示・管理する。管理者のみ利用可能。

### レイアウト

```
┌──────────────────────────────────────────────────────┐
│ 営業日報  日報一覧 / 顧客マスタ / ユーザーマスタ     │
├──────────────────────────────────────────────────────┤
│ ユーザーマスタ                    [+ ユーザーを追加する] │
│                                                      │
│  氏名 [        ]  ロール [全員 ▼]  [検索]           │
│                                                      │
│  氏名         メールアドレス    ロール   上長      操作         │
│  ──────────────────────────────────────────────────  │
│  山田 太郎   yamada@co.jp  営業     鈴木一郎  [編集] [削除] │
│  鈴木 花子   suzuki@co.jp  営業     鈴木一郎  [編集] [削除] │
│  鈴木 一郎   ichiro@co.jp  上長     （なし）  [編集] [削除] │
│  ...                                                 │
│                                                      │
│  < 1 2 3 ... >                                       │
└──────────────────────────────────────────────────────┘
```

### 表示項目

| 項目名 | 内容 |
|--------|------|
| 氏名 | ユーザーの氏名 |
| メールアドレス | ログインID兼連絡先 |
| ロール | `営業` / `上長` / `管理者` |
| 上長 | 紐付いている上長の氏名（未設定の場合は「なし」） |
| 操作 | 編集ボタン / 削除ボタン |

### アクション

| アクション | 処理 |
|------------|------|
| ユーザーを追加するボタン | ユーザーマスタ編集画面（新規）へ遷移 |
| 編集ボタン | ユーザーマスタ編集画面（編集）へ遷移 |
| 削除ボタン | 確認ダイアログ表示後、削除実行。日報が存在するユーザーは削除不可（エラー表示） |

## 11. ユーザーマスタ編集画面 (USER-EDIT)

### 概要

ユーザー情報を新規登録または編集する。管理者のみ利用可能。

### レイアウト

```
┌──────────────────────────────────────────────────────┐
│ 営業日報                                             │
├──────────────────────────────────────────────────────┤
│ ユーザーマスタ編集                                   │
│                                                      │
│  氏名 *                                              │
│  [                                               ]   │
│                                                      │
│  メールアドレス *                                    │
│  [                                               ]   │
│                                                      │
│  パスワード（新規登録時必須 / 変更時のみ入力）       │
│  [                                               ]   │
│                                                      │
│  ロール *                                            │
│  [営業 ▼                                         ]   │
│                                                      │
│  上長                                                │
│  [（未設定） ▼                                   ]   │
│                                                      │
│            [キャンセル]  [保存する]                  │
└──────────────────────────────────────────────────────┘
```

### 入力項目

| 項目名 | 種別 | 必須 | バリデーション |
|--------|------|------|----------------|
| 氏名 | テキスト | ○ | 1文字以上、50文字以内 |
| メールアドレス | テキスト (email) | ○ | メール形式、システム内で一意 |
| パスワード | パスワード | 新規登録時のみ○ | 8文字以上 |
| ロール | セレクトボックス（`営業` / `上長` / `管理者`） | ○ | 選択必須 |
| 上長 | セレクトボックス（ロールが「上長」または「管理者」のユーザーから選択） | - | - |

### アクション

| アクション | 処理 |
|------------|------|
| 保存するボタン | バリデーション後、保存 → ユーザーマスタ一覧画面へ遷移 |
| キャンセルボタン | ユーザーマスタ一覧画面へ戻る |
